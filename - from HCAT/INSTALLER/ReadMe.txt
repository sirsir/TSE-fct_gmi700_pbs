=== CLIENT Software

After install "gmi700_pbs_client_setup.msi", 
copy and replace "gmi700_pbs_client.exe.config" in "C:\Program Files (x86)\TSE\GMI700 Panel Barcode System Client".


=== SERVER Software

After install "gmi700_pbs_setup.msi", 
1. copy and replace "gmi700_pbs_server.exe.config" in "C:\Program Files (x86)\TSE\GMI700 Panel Barcode System Server\bin".
2. copy and replace "config.ini" in "C:\ProgramData\gmi700_pbs_server\gmi700_pbs_server\1.0.0.0".
